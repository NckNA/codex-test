Описание раздела

Раздел «Вакансии» предназначен для работодателей и соискателей. Здесь работодатели смогут размещать детальные объявления о работе, а соискатели – создавать резюме и откликаться на вакансии. В будущем будет реализован функционал фильтрации по отрасли, уровню зарплаты, типу занятости, опыту работы и другим параметрам.

Для работодателей

В этой версии портала employers смогут:

  • Создать корпоративный профиль компании
  • Разместить вакансию с требуемыми навыками и условиями
  • Управлять откликами соискателей

Будущие улучшения включают CRM‑систему для отслеживания кандидатов и шаблоны вакансий для разных типов позиций.

Для соискателей

Соискатели смогут создать подробное резюме, загрузить документы и управлять несколькими версиями резюме для разных позиций. В будущем будет реализована система рекомендаций вакансий на основе профиля кандидата.
